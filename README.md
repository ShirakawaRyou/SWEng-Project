使用方式：
在Google extension中加载已解压的扩展程序
文件夹为chromeExtension

后端 Python + Django
在backend文件夹下使用 python manage.py runserver

前端 Node.js+Vue3
在frontend文件夹下使用 npm run serve

在chrome中右上角的插件目录中点击按钮即可跳转到localhost:8080/IndexPage页面

页面中为后端输出的内容
